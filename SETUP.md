⚙️ Setup Steps
**1. Clone the Repository**

git clone <repository-url>
cd portfolio-project

•Navigate into the project directory.

**2.Open Locally**

• Double-click index.html to open in any modern browser.
• All styling (style.css) and interactivity (script.js) will load automatically.

**Deployment**

**1.GitHub Pages**

• Push the project to a GitHub repository.
• Enable GitHub Pages in repository settings.
• Your portfolio will be live at https://Logicwithdevanshi.github.io/FUTURE_FS_01/.

**2.Netlify / Vercel** 

• Drag and drop the project folder into Netlify or Vercel.
• Alternatively, connect your GitHub repo for automatic deployment.
• Other Hosting Options

• Any static hosting service (e.g., Firebase Hosting, Surge, Render) works.
• Upload the project folder and point to index.html.
